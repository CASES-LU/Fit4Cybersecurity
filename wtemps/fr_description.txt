Description

Cet outil d'auto-évaluation est nommé Fit4Cybersecurity pour souligner l'introduction à la sécurité de l'information. Il est aussi un prérequis pour faire un Diagnostic CASES: un score minimal de 80/100 est requis. Le but pour cette restriction nous donne la possibilité pour mieux adapter les recommendations à votre organisation.

Pendant cet auto-evaluation $$qnumber$$ questions à choix multiples vous ont été posées sur lesquels vous avez réponue. Après avoir données ces réponses, votre résultat avec des recommandations ont été afficher.

Finalement, vous avez télécharger ce document en français et vous pouvez bien sûr aussi le télécharger en anglais et allemand ce qui vous aidera pour communiquer les résultats à vos fournisseur si besoin.
